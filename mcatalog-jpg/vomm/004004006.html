Up to 110Pa External Static Pressure

High-rise buildings are usually designed to have outdoor units installed inside them using ductwork, because the height difference to the rooftop is too big. The DVM S2 has up to 110Pa External Static Pressure*, which ensures that it can discharge air effectively through a much longer duct. So, it gives you an even greater choice when selecting an installation location inside the building.
