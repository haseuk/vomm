Q 08

Is the DVM S2 compatible with the
existing indoor or outdoor units of a
conventional DVM S system?

"The DVM S2 is compatible with the existing indoor units and controllers of the DVM S, but it should not be used with the existing DVM S outdoor units on the same system."

The DVM S2 is fully compatible with existing indoor units and controllers that have been used with DVM S, and the indoor units and controllers for DVM S2 are also compatible with the existing DVM S outdoor units.

However, the DVM S2 and the DVM S outdoor units should not be used together in the same system module, although they might still work and it is possible that there may be no error in communicating between them. In particular, the latest innovative functions of the DVM S2 are not available when installed together with the DVM S. So, Samsung advises not to mix them in the same system and does not provide any official technical guidelines for a mixed installation.
